# CS2 Sens Tuner & RPG Engine

## Description
This is my first Python project. It includes:
* **Sensitivity Tuner**: A tool to help configure CS2 mouse sensitivity.
* **RPG Engine**: A console-based game with a level-up system and persistent saves.

## Features
* Save/Load system using file handling.
* Error handling with try/except.
* Interactive menu with while loops.
